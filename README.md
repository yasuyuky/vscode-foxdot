# vscode-foxdot README

vscode extension for [FoxDot](http://foxdot.org)

## Features

- `foxdot.start`
- `foxdot.send`
- `foxdot.stop`
- `foxdot.restart`
- `foxdot.record`
- `foxdot.stopRecording`

## Requirements

- [FoxDot](http://foxdot.org)

## Extension Settings

- `foxdot.pythonPath`
  - Python Path with FoxDot installed (Optional)
